# Online Quiz Examination System

## Project Overview

The Online Quiz Examination System is a simple Python-based console application developed using Object-Oriented Programming concepts. The project allows users to answer multiple-choice questions, calculate scores, and display performance results.

This project is useful for beginners to understand Python classes, objects, loops, condition statements, and user interaction.


# Features

- Multiple-choice quiz questions
- Automatic score calculation
- Performance evaluation
- User-friendly console interface
- Object-Oriented Programming implementation
  
# Technologies Used

- Python
- OOP Concepts
- VS Code
